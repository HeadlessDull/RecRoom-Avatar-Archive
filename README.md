# RecRoom-Avatar-Archive
Massive collection of blender ready RecRoom Avatar Assets!

# This is still W.I.P
**All below is a currently ongoing project that is continuly evlolving to remove friction and continue adding missing assets!**


# The Archive
A large dump of images/textures/and fbx's of most if not all Avatar items used in game.
> without UGC items from the community ⚠️
> > We could aceppt **PERSONALLY** include given UGC assets *later* on.

>  Not *"yet"* blender ready out of the box ⚠️
> > Tools are coming to help with that, keep updated.


# Current Blender Tools
